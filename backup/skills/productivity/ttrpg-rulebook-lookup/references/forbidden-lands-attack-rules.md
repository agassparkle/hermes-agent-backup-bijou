# Forbidden Lands — condensed attack lookup

Source: Forbidden Lands - Player's Handbook, combat & damage rules.

## Dice for attacks

- PC melee/ranged attack roll = Skill Dice + Gear/Weapon Dice
- Skill Dice = skill level + linked attribute
  - Melee -> attribute tied to MELEE
  - Ranged -> attribute tied to MARKSMANSHIP
- Gear Dice = weapon Gear Bonus
- Do not use Base Dice for PC attack rolls

## Melee attack

- Declare action: SLASH, STAB, PUNCH/KICK
- Roll skill + weapon dice
- Each x = hit
- Damage = Weapon Damage + 1 per extra x
- Close combat range is usually NEAR; Arm's Length is a move
- Hidden combinations let both sides choose a hidden attack/defense action before the exchange

## Ranged attack

- Use MARKSMANSHIP + weapon Gear Bonus
- Must READY first unless FAST SHOOTER talent allows otherwise
- Then AIM (+1, same round only) then SHOOT (slow action)
- Crossbows must be reloaded
- Target can PARRY or DODGE to eliminate one x each
- Range mods: NEAR -2, SHORT -1, LONG +1

## Ambush / sneak

- Opposed SNEAK roll before acting
- Success gives free action; target cannot PARRY or DODGE the following attack
